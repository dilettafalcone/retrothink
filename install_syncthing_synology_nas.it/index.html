Installare Syncthing su un NAS Synology

2023-04-05 · Sync

  1. Apri Package Center

  2. Installa Docker

  3. Apri File Station

  4. Apri la cartella ‘docker’ sulla sinistra e crea al suo interno 3 nuove cartelle chiamate syncthing, data1, data2

  5. Apri Control Panel, apri Task Scheduler, clicca su Create -> Scheduled Task -> User-defined script

  6. General:

    • Task: Installazione Syncthing
    • User: root
    • Togli la spunta da ‘Enabled’

  7. Schedule:

    • Spunta ‘Run on the following date’, lascia la data di oggi, ‘Do not repeat’

  8. Task settings:

    • Spunta ‘Send run details by email’ e inserisci la tua mail
    • In ‘Run command’ inserisci questo testo, modificandolo secondo i tuoi valori di PUID, PGID, Time Zone (TZ) e volumi:
    docker run -d --name=syncthing \
-p 8384:8384 \
-p 22000:22000/tcp \
-p 22000:22000/udp \
-p 21027:21027/udp \
-e PUID=1026 \
-e PGID=100 \
-e TZ=Europe/Rome \
-v /volume1/docker/syncthing:/config \
-v /volume1/docker/data1:/data1 \
-v /volume1/docker/data2:/data2 \
--restart always \
ghcr.io/linuxserver/syncthing

  9. Clicca OK sull’avviso che appare

  10. Clicca col tasto destro su ‘Installazione Syncthing’ che hai creato e clicca su ‘Run’

  11. Aspetta un po’ poi vai su http://:8384

  12. Ti apparirà un avviso, vai su Impostazioni

    • Generale: inserisci il nome del tuo NAS nella voce Nome Dispositivo
    • Interfaccia Grafica Utente: scegli un username e una password

  13. Segui la guida su come si usa Syncthing
