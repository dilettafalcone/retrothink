Organizzare i PDF con Zotero

Cos’è Zotero e come installarlo

Zotero è utilissimo per organizzare i propri PDF.

Debian/Ubuntu

sudo apt install curl -y

curl -sL https://raw.githubusercontent.com/retorquere/zotero-deb/master/install.sh | sudo bash

sudo apt update

sudo apt install zotero

Arch Linux

Installa yay

yay zotero-bin

Crea un account gratuito su Zotero

Vai a Preferenze, Sincronizzazione e crea un account.

Utilizzare Syncthing per sincronizzare i dati di Zotero

Installa Syncthing

Aggiungi una cartella mettendo come percorso:

/home/tuonomeutente/Zotero/storage/

e sincronizzala con tutti i tuoi computer su cui hai installato Zotero

In Zotero, vai su Preferenze, Sincronizzazione, inserisci nome utente e password di Zotero, spunta Sincronizza automaticamente, togli entrambe le spunte dalle opzioni di sincronizzazione dei file, se ne occuperà Syncthing! Adesso Zotero sincronizzerà il database e Syncthing tutti i file allegati.

Utilizzare WebDAV per sincronizzare i dati di Zotero

Puoi usare un WebDAV Server per sincronizzare i tuoi dati in Zotero su tutti i tuoi dispositivi. In questo caso utilizzeremo un NAS Synology.

Dalla pagina principale del tuo Synology vai su Packet Center e installa “WebDAV Server”

Apri e spunta “Enable HTTP” e “Enable HTTPS”, salva e applica.

Apri File Station e nella cartella home (/homes/tuonomeutente) crea una cartella chiamata “zotero”

Apri l’applicazione di Zotero (su Android utilizzo con successo Zoo for Zotero) ed entra nelle Preference, vai su Sincronizzazione e spunta solamente “Sincronizza gli allegati nella biblioteca personale con” scegliendo “WebDAV”, ad url metti

http tuoindirizzoNAS:5005/homes/tuonomeutente

Inserisci il Nome utente e Password che usi per il tuo NAS Synology
