GrapheneOS: riprendere il controllo del proprio telefono

Cos’è GrapheneOS

GrapheneOS è un sistema operativo open source basato su Android Open Source Project (AOSP), incentrato sulla sicurezza e sulla privacy. Il progetto è stato fondato da Daniel Micay nel 2014.

Telefoni ufficialmente supportati

  • Pixel 7 Pro (cheetah)
  • Pixel 7 (panther)
  • Pixel 6a (bluejay)
  • Pixel 6 Pro (raven)
  • Pixel 6 (oriole)
  • Pixel 5a (barbet)
  • Pixel 5 (redfin)
  • Pixel 4a (5G) (bramble)
  • Pixel 4a (sunfish)

Installazione tramite Web installer

Il sistema operativo si installa facilmente sul telefono tramite il Web Installer, basta collegare al pc il vostro telefono Pixel con un cavo di buona qualità (il primo cavo dati che ho usato non funzionava) e seguire le istruzioni.
